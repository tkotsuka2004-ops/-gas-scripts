# gas-scripts

Google Apps Script (GAS) のスクリプトを管理するプロジェクトです。

## 概要

- 用途: Google Apps Script で作成する各種スクリプトの管理
- 言語: JavaScript (Google Apps Script)

## ディレクトリ構成

プロジェクトの成長に合わせて、スクリプトごとにサブディレクトリを分けて管理する想定です。

## 開発メモ

- clasp (Command Line Apps Script Projects) を使う場合は、`.clasp.json` や `appsscript.json` はプロジェクトごとに管理する
- 認証情報やスクリプトID等の秘匿情報はコミットしない
